# 灵可社区 (Lingke Shequ)

基于区块链的互助生态系统网站，展示封闭邀请制社区的委托互助机制、币池奖励系统和社区积分体系。

## 项目简介

灵可社区是灵可生态系统的重要组成部分：
- **善行天下**：记录善行，积累功值
- **灵可币**：基于BSC的代币，用于社区委托
- **灵可社区**：封闭邀请制互助平台
- **灵可世集**：元宇宙本地生活服务平台

## 功能特性

### 1. Hero 主视觉区
- 动态流体背景效果（Canvas动画）
- 3D文字入场动画
- 展示社区核心理念：封闭邀请制 · 区块链互助生态
- 关键数据展示：500+功值门槛、100%互助委托

### 2. 社区机制区
三大核心机制：
- **邀请准入**：功值达500获得邀请资格
- **委托互助**：使用灵可币发起/接受委托
- **币池奖励**：完成委托获得奖励

### 3. 委托流程区
- 发起委托 → 设定币池金额
- 接受委托 → 执行任务内容
- 获得奖励 → 自动发放+产生积分

### 4. 社区积分区
- 3D旋转水晶视觉元素
- 积分获取方式说明
- 权限解锁体系展示

### 5. 社区模拟页面
点击"探索社区"可进入模拟场景：
- 100个生活化委托场景
- 等待接受/进行中/已完成三栏展示
- 币池动态实时统计
- 分类筛选功能

## 技术栈

- **框架**: React + TypeScript + Vite
- **样式**: Tailwind CSS
- **组件库**: shadcn/ui
- **动画**: GSAP (ScrollTrigger)
- **路由**: React Router DOM
- **图标**: Lucide React

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 设计特点

- **品牌色彩**：深黑(#1a1a1a) + 温暖米白(#e9e4df) + 柔和金色(#c9a87c)
- **动效系统**：GSAP滚动动画、流体背景、3D变换
- **响应式设计**：完美适配桌面端和移动端
- **交互体验**：磁性按钮、悬停效果、平滑滚动

## 与生态系统关联

- 用户需通过 **善行天下** 积累功值
- 使用 **灵可币** 进行委托互助
- 积分可在 **灵可世集** 解锁服务商权限

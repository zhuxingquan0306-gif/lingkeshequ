## 🌟 项目简介

灵可社区是一个**封闭邀请制**的互助委托平台。正式用户通过使用灵可币来发起委托，其他用户接受委托并完成，获得对应委托币池里的灵可币奖励。

### 核心特色

- 🔒 **封闭邀请制** - 功值达到500才能加入
- 🤝 **互助委托** - 发起委托、接受委托、共同完成
- 💰 **币池机制** - 委托币池由发起者设定，完成即获奖励
- 💎 **社区积分** - 灵可币使用产生积分，是权限基础

## 🚀 快速开始

### 方式一：直接打开

由于本项目是纯静态网站，您可以直接在浏览器中打开 `index.html` 文件。

### 方式二：使用本地服务器

```bash
# 克隆仓库
git clone https://github.com/zhuxingquan0306-gif/lingkeshequ.git

# 进入项目目录
cd lingkeshequ

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或启动生产服务器
npm start
```

服务器启动后，访问 http://localhost:3000 即可查看网站。

## 🎨 页面详情

### 主页面 (index.html)

主页面包含以下区块：

1. **导航栏** - 动态效果导航，支持平滑滚动
2. **主视觉区** - 深色背景配粒子动画，展示社区理念
3. **数据统计** - 展示用户数、完成委托数、币流通量、完成率
4. **如何运作** - 四步流程：发起委托 → 接受委托 → 完成委托 → 获得奖励
5. **委托广场** - 展示各类委托，支持筛选
6. **社区积分** - 积分机制说明与可视化
7. **加入入口** - 功值500门槛说明
8. **页脚** - 品牌信息与快速链接

## 🔄 委托流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  发起委托   │ →  │  接受委托   │ →  │  完成委托   │ →  │  获得奖励   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      │                  │                  │                  │
      ▼                  ▼                  ▼                  ▼
  使用灵可币         浏览委托列表        按要求完成         获得币池
  设定币池           选择感兴趣的        提交证明           灵可币
```

### 委托类型

- 👤 **个人委托** - 针对自己的需求
- 👥 **他人委托** - 帮助他人完成需求
- 🌍 **事物委托** - 针对非人事物（如社区维护、动物救助等）

## 💎 社区积分机制

### 积分获取
- 使用灵可币发起委托 → 获得积分
- 使用灵可币接受委托 → 获得积分
- 完成委托 → 额外积分奖励

### 积分用途
- **500+ 功值** - 获得灵可社区邀请资格
- **积分等级** - 解锁更多社区权益
- **信誉体系** - 积分代表社区贡献度
